Tags

<article>

La etiqueta <article> se usa para contenido independiente que podría distribuirse o reutilizarse, como una entrada de blog, una noticia o un comentario.


                    <article>
                        <h2>Noticias del día</h2>
                        <p>Hoy se lanzó una nueva versión de HTML...</p>
                    </article>

<section>

<section> agrupa contenido relacionado bajo un mismo tema. Puede contener encabezados, párrafos, artículos, etc.


                    <section>
                        <h2>Servicios</h2>
                        <p>Ofrecemos desarrollo web, diseño UI/UX y SEO.</p>
                    </section>

<nav>

<nav> define una sección con enlaces de navegación. Usualmente se encuentra en menús, barras laterales o encabezados.


                    <nav>
                        <ul>
                            <li><a href="index.html">Inicio</a></li>
                            <li><a href="contact.html">Contacto</a></li>
                        </ul>
                    </nav>

<header>

<header> representa la cabecera de una página o una sección. Suele contener el logo, el título y la navegación principal.


                    <header>
                        <h1>Mi blog personal</h1>
                        <nav>...</nav>
                    </header>

<footer>

<footer> define el pie de una página o sección. Suele incluir información de contacto, derechos de autor, o enlaces útiles.


                    <footer>
                        <p>© 2025 Mi sitio web. Todos los derechos reservados.</p>
                    </footer>
